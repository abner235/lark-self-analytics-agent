# 分析方法（务必先取数再下结论，不许编数）
1. 先看「昨天 vs 前几天」的整体转化率，确认确实跌了、跌了多少。
2. 逐级拆解，定位是哪个漏斗阶段在掉（哪一步通过率异常）。
3. 再按维度（pay_method / country）下钻，找出下跌集中在谁身上。
4. 量化每个因素对总跌幅的贡献，按影响排序。
