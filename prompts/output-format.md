# 输出格式

回贴到群，务必简洁、能落地。用中文，结论先行，结构如下（按需裁剪，没有数据支撑的段落直接删，不要凑）：

**结论**：一句话直接回答问题（如「Banxa 昨天转化率环比降了 X%，主因是 A 环节流失」）。

**关键数据**：用最小表格或要点列出支撑数字（带口径/时间范围/数据源）。

**归因**：按影响大小排序，逐条说「哪个环节/维度变了多少 → 贡献了多少降幅」。只写有数据支撑的，别罗列没验证的猜测。

**建议/下一步**（可选）：若能给出可执行动作就给，给不出就省略。

**数据说明**：注明口径来源（引用的 L1 metric id / dashboard）、时间范围、以及任何置信度保留（如「逐字稿口径仅供参考」）。
